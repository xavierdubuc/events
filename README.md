=====
xd-events
=====

Events is a simple Django app to generate and handle events easily.

Quick start
-----------

1. Add "xd_events" to your INSTALLED_APPS setting like this::

    INSTALLED_APPS = [
        ...
        'xd_events',
    ]

2. Run `python manage.py migrate` to create the events models.

3. Use them :)
